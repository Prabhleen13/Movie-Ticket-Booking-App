# 🎬 Movie Ticket Booking App

A Java-based application that allows users to book movie tickets efficiently through a simple interface. This project demonstrates core concepts of Java, JDBC, and MySQL database integration.

---

## 📌 Project Description

The Movie Ticket Booking App is designed to simulate an online ticket booking system where users can browse movies, select seats, and confirm bookings. The application connects to a MySQL database to manage user data, movie details, and booking records.

---

## ✨ Features

- 🔐 User Registration & Login
- 🎥 View Available Movies
- 🪑 Seat Selection System
- 🎟️ Ticket Booking Functionality
- 💾 Data Storage using MySQL
- ❌ Booking Validation & Error Handling

---

## 🛠️ Technologies Used

- **Java** (Core Java)
- **JDBC** (Java Database Connectivity)
- **MySQL** (Database)
- **MySQL Workbench** (Database Management)

## ⚙️ How to Run the Project

### 1. Clone the Repository


### 2. Open in IDE
- Open the project in any Java IDE (Eclipse / IntelliJ / NetBeans)

### 3. Setup Database
- Open MySQL Workbench
- Create a new database (e.g., `movie_booking`)
- Import the SQL file from the `database/` folder

### 4. Configure Database Connection
- Update your database credentials in the Java code:
  - Username
  - Password
  - Database name

### 5. Run the Application
- Run the main Java file to start the application

---

## 🗄️ Database Setup

The database file is provided in the `database/` folder.

Steps:
1. Open MySQL Workbench  
2. Go to Server → Data Import  
3. Select the `.sql` file  
4. Execute to create tables and insert data
